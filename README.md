# SystemDesign.dev

An interactive Next.js web application that teaches developers how major tech systems are architected through clean visual diagrams, step-by-step animated flows, and plain-language explanations.

## Systems Covered

15 systems including ChatGPT, Google Search, Uber ETA, Amazon S3, YouTube, Kafka, WhatsApp, Spotify, Slack, Reddit, Bluesky, Twitter Timeline, URL Shortener, Payment System, and Stock Exchange.
